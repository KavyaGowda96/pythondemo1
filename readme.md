# python.py

This repository contains a simple Python script, `python.py`, with several example functions and code snippets.

## Contents

- `pal(str)` - checks whether a given string is a palindrome by reversing it and comparing it to the original.
- `second_largest(arr)` - returns the second largest number in a list.
- A word frequency counter for the sample string `"hello word"`.
- A basic sum-pairs example that reads input values and a target number.

## Running the script

The script can be executed with Python 3:

```bash
python python.py
```

Note: `python.py` currently includes multiple `if __name__ == '__main__'` blocks, so only the last one will run when executed directly.
